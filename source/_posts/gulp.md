## Gulp


安装gulp

``` bash
	npm i -g gulp
```
初始化

``` bash
	npm init
```
当前文件中安装并保存在配置文件中

``` bash
	npm i --sace-dev gulp
```
模块安装

``` bash
	npm i --save-dev gulp-clean gulp-concat gulp-connect gulp-cssmin gulp-imagemin gulp-less gulp-load-plugins gulp-plumber gulp-uglify open
```
